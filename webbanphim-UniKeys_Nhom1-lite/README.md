# UniKeys

Dự án PHP/MySQL mô phỏng website bán bàn phím cơ, keycap, switch và phụ kiện custom keyboard.

- Giao diện người dùng: trang chủ, tìm kiếm, chi tiết sản phẩm, giỏ hàng, thanh toán
- Trang quản trị: sản phẩm, thương hiệu, nhà phân phối, nhà cung cấp, đơn hàng, cửa hàng
- Dữ liệu mẫu: bàn phím cơ, keycap, switch, phụ kiện mod và combo starter

Lưu ý: tên cơ sở dữ liệu trong mã nguồn vẫn giữ là `webbanphim` để tránh phải sửa nhiều logic cũ.
